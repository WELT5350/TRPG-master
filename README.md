## 使用规则

- 日常查看和运行项目时使用 `main`。
- 查找个人历史材料时切换到本归档分支。
- 需要核验文件整理前的原始位置或内容时查看原貌标签。
- 本分支原则上只读；后续归档使用新的日期分支，不在旧归档上持续开发。
- 归档材料不自动视为当前有效需求，采用其中内容前应重新确认。

## 后续归档命名

- 精确快照标签：`archive/personal-main-YYYY-MM-DD`
- 可浏览归档分支：`archive/personal-docs-YYYY-MM-DD`

这样可以让 `main` 保持干净，同时让个人探索、历史决策和参考资料有清晰留痕。
