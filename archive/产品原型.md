# AI 角色扮演平台 · 产品定义

状态：proposal  
版本：v0.1
日期：2026-07-08  

---

## 1. 一句话定义

一个 **AI 驱动的多人角色扮演平台**：玩家可以自由游玩预设或自定义的剧情模组，由多 Agent 协同的 AI 主持游戏，并辅以 AIGC 图片生成、世界观构建工具和内置合规审查，以网页形式提供沉浸式、低门槛的角色扮演体验。

---

## 2. 目标用户

- **新手玩家**：想体验 TRPG/剧本杀/角色扮演但缺乏主持和规则经验的小团体或个人。
- **资深玩家/主持人**：希望借助 AI 辅助创作、减少繁琐准备工作的创作者。
- **世界观作者**：想将自创世界转化为可游玩模组的写作者。
- **社交群体**：需要集体娱乐活动的学生社团、团建团队。

---

## 3. 核心产品形态

- **网页应用**（PC / 手机自适应），而非原生 App。
- **游戏形式**：以文字叙事为主，辅以场景图、角色立绘、线索卡片等视觉元素。
- **主持方式**：AI 集群自动主持，无需真人 KP/DM。
- **内容来源**：平台预置模组 + 用户使用创作工具自建模组。

---

## 4. 多 Agent 架构（核心差异化）

AI 主持由至少四个专职 Agent 协同工作：

| Agent | 职责 |
|-------|------|
| **裁决 Agent** | 判断玩家意图是否合法、是否触发检定、设定难度 |
| **规则 Agent** | 执行确定性运算（骰点、数值结算、状态变更、查表） |
| **叙事 Agent** | 生成场景描述、NPC 对话、剧情推进文本 |
| **合规 Agent** | 实时审查所有输入输出，拦截违规内容（涉政、涉黄、暴力等） |

**原则**：确定性运算走规则 Agent，创造性叙事走叙事 Agent，两者解耦，保证稳定与合规。

---

## 5. 功能模块（首发范围）

### 5.1 玩家端
- 模组大厅（浏览、筛选、开始游戏）
- 房间系统（创建/加入房间，角色分配）
- 游戏主界面（场景图 + 叙事流 + 输入栏 + 状态条）
- 角色卡与线索板（抽屉式展示）
- 关键选择/投票（多人分歧点）
- 结局复盘（时间线、决策树、错过内容）

### 5.2 AI 主持端
- 支持自由文本输入（玩家可随意发言）
- 自动识别行动意图（调查、交谈、移动、使用物品等）
- 触发技能检定（显示骰值、难度、结果）
- 私密信息分发（仅特定玩家可见）
- 动态场景切换与氛围描述

### 5.3 AIGC 图片生成
- 在场景切换、关键 NPC 登场、重要线索发现时自动生成插画
- 提供“重绘”按钮供玩家手动触发
- 图片基于场景元数据（地点、时间、氛围）生成，保持风格统一

### 5.4 创作工具（基础版）
- **角色创建**：快速建卡（属性、技能、背景）或随机生成
- **世界观构建**：输入自定义设定（如“蒸汽朋克魔法城市”），系统生成初始场景和任务
- **模组设计**（后续扩展）：拖拽式场景编辑、事件触发器配置

### 5.5 合规与安全
- 输入层敏感词过滤
- 输出层合规 Agent 实时审查，违规自动替换为安全文本并记录日志
- 所有 AI 生成内容带时间戳，不可回溯修改（存档点除外）

---

## 6. 关键设计决策

| 决策点 | 选择 | 理由 |
|--------|------|------|
| 架构 | 多 Agent 分工 | 稳定可控，各司其职，便于优化和容错 |
| AI 文本生成 | 开源模型 + 微调 + 提示词工程 | 平衡创意与可控性，先提示词验证，再收集数据微调 |
| AIGC 图片 | 关键节点自动生成 + 玩家手动重绘 | 控制成本，保持沉浸感 |
| 时间戳与回溯 | 不可回溯，但提供存档分支 | 模拟真实跑团不可逆性，增加重复游玩价值 |
| 合规审查 | 输入+输出双重过滤 + 独立 Agent | 全面覆盖，满足商业化合规要求 |
| UI 风格 | 风格化主题（如哥特/奇幻），以故事流为中心 | 突出沉浸感，区别于工具式 VTT |

---

## 7. 明确不做（首期）

- 3D 场景或实时动画
- 实时语音交互（以文字为主）
- 完全无边界的沙盒自由模式
- 多人同时编辑世界观/模组
- 直接导入外部 PDF/Word 模组
- 付费订阅与商城（首发免费）

---

## 8. 核心信息结构（简略）

- **用户**：账号、角色库、游玩历史、创作作品
- **模组**：元数据、场景图、NPC、事件触发器、结局、美术资源
- **游戏房间**：玩家列表、AI Agent 状态、运行时数据、时间戳流水
- **世界观**：设定集（历史、地理、种族、规则），可被多个模组共享

## 9. 下一步建议

1. 团队评审本产品定义，确认方向与范围。
